Explorando Arrays em Java na Área Financeira

Em Java, um array é como uma carteira de Ações, onde cada slot representa um ativo. Ele é utilizado para armazenar coleções de valores do mesmo tipo, permitindo o acesso e manipulação eficientes desses ativos.

Neste exemplo, criamos um array de números inteiros. Cada número é como um ativo único na carteira, e podemos acessá-los facilmente utilizando seus índices.
